## 🧠 Projets ML & Data Science

### 📊 [Amazon Sales Analysis & Prediction](https://lnkd.in/dVxGXkU3)
**Analyse complète et prédiction des ventes Amazon**
- **Objectif** : Exploration, analyse et prédiction des ventes e-commerce avec données 2025
- **Techniques appliquées** :
  - Exploratory Data Analysis (EDA) complète
  - Nettoyage de données et traitement des valeurs manquantes
  - Visualisations avancées (évolution, saisonnalité, tendances)
  - Modélisation prédictive multi-algorithmes
- **Modèles & Performances** :
  - Régression Linéaire → RMSE = 0.25, R² ≈ 0.94
  - Arbre de Décision → RMSE = 0.24
  - Forêt Aléatoire → RMSE = 0.19, R² ≈ 0.97
  - Gradient Boosting → RMSE = 0.17, R² ≈ 0.97 ⭐ (Best)
- **Stack** : Python, Pandas, NumPy, Matplotlib, Seaborn, scikit-learn
- **Insights** : Importance du prétraitement, comparaison modèles, méthodes d'ensemble supérieures

### 🧠 [BERT Sentiment Analysis](https://lnkd.in/d_-GZCFx)
**Analyse de Sentiment avec BERT - Apprentissage en 60 jours**
- **Objectif** : Fine-tuning BERT pour classification multi-sentiments (avis clients)
- **Étapes clés** :
  1. Architecture BERT de zéro
  2. Nettoyage de données (HTML, espaces, normalisation)
  3. Tokenisation et encoding avec `encode_plus`
  4. Gestion troncature/padding et masques d'attention
  5. DataLoaders PyTorch (train/validation)
  6. Fine-tuning modèle pré-entraîné (Hugging Face)
  7. Optimiseur + scheduler de learning rate
  8. Accélération GPU via CUDA
- **Résultats** : Modèle performant prêt pour déploiement production
- **Stack** : PyTorch, Transformers (Hugging Face), CUDA, scikit-learn

### 🏥 [Breast Cancer Detection - MLOps Pipeline](https://lnkd.in/eWWrrXUU)
**Classification Tumeurs Mammaires avec approche MLOps**
- **Dataset** : 569 patients, 30 features morphologiques
- **Distribution** : 62.7% bénignes vs 37.3% malignes (déséquilibre modéré)
- **Analyse Exploratoire** :
  - Feature importance : "worst" features (concave_points_worst ρ=0.794)
  - Détection multicollinéarité (27 paires |corr| > 0.9)
  - Conservation intentionnelle des outliers (cas cliniques atypiques)
- **Pipeline ML** :
  - Feature engineering pour relations non-linéaires
  - Standardisation StandardScaler
  - Split stratifié train/test (80/20)
  - Sélection features pour interprétabilité/performance
- **MLOps (Phase 2)** :
  - Versioning données/modèles (DVC)
  - Multi-model training + hyperparamètre optimization
  - Métriques médicales (Recall, ROC-AUC, Precision)
  - Validation tests et monitoring production
- **Stack** : Python, Pandas, scikit-learn, Seaborn, Matplotlib
